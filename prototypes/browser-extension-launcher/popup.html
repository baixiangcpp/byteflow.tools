Byteflow Tools

Launcher prototype. It opens existing tools and requests no clipboard, host, storage, or history permissions.

JSON Formatter Format, validate, and inspect JSON locally. Base64 Encode/Decode Encode or decode Base64 text in the browser. JWT Decoder Inspect JWT headers and payloads without verification calls. Regex Tester Test expressions and match groups locally. URL Encode/Decode Encode and decode query-safe text. UUID Generator Create UUIDs for fixtures and tests. Hash Generator Generate digest values in the browser. Log Scrubber Remove common tokens and secrets before sharing logs.
